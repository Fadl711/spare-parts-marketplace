# صفحة الملف الشخصي للبائع - دليل الاستخدام

## نظرة عامة

تم إنشاء صفحة ملف شخصي مخصصة للبائعين في لوحة التحكم، تتيح لهم إدارة جميع معلومات متجرهم بسهولة.

## المميزات

### 1. معلومات المتجر

- **اسم المتجر**: الاسم الذي يظهر للعملاء
- **اسم المالك**: اسم صاحب المتجر
- **شعار المتجر**:
  - رفع صورة مع محرر صور مدمج
  - قص دائري تلقائي
  - الحجم المقترح: 500x500 بكسل

### 2. معلومات الاتصال

- **رقم الهاتف**: مع التحقق من التفرد
- **البريد الإلكتروني**: اختياري
- **رابط واتساب**: للتواصل السريع مع العملاء

### 3. الموقع

- **المدينة والحي**: لتحديد موقع المتجر
- **العنوان التفصيلي**: وصف كامل للموقع
- **الإحداثيات الجغرافية** (اختياري):
  - خط العرض (Latitude)
  - خط الطول (Longitude)
  - للظهور على الخرائط في التطبيق

### 4. ساعات العمل

- نظام مرن لإضافة أيام وأوقات العمل
- يمكن إضافة عدة أيام
- يتم حفظها كـ JSON في قاعدة البيانات

### 5. معلومات الاشتراك

- عرض تاريخ انتهاء الاشتراك
- حقل للقراءة فقط
- للتجديد يجب التواصل مع الإدارة

### 6. تغيير كلمة المرور

- **كلمة المرور الحالية**: للتحقق من الهوية
- **كلمة المرور الجديدة**: مع تأكيد
- الحد الأدنى: 8 أحرف
- التشفير تلقائي باستخدام Hash

## الوصول للصفحة

```
URL: /seller/profile
Route Name: filament.seller.auth.profile
```

## الأمان

- ✅ التحقق من كلمة المرور الحالية قبل التغيير
- ✅ تشفير كلمات المرور باستخدام Hash::make()
- ✅ التحقق من تفرد رقم الهاتف والبريد الإلكتروني
- ✅ استخدام Filament Guard (seller_web)

## ملاحظات تقنية

- يستخدم `Filament\Schemas\Schema` بدلاً من `Form`
- يمتد من `Filament\Auth\Pages\EditProfile`
- يستخدم `Halt` exception لإيقاف الحفظ عند وجود أخطاء
- الصور تُحفظ في `storage/app/public/store-logos`
